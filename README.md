
# QA API Testing Project – Restful Booker

## Project Overview
Manual API testing project for Restful Booker API, including Postman collection, test cases, test run results, and bug reports.

## Scope of Testing
- Authentication
- Booking creation
- Booking retrieval
- Booking update
- Partial booking update
- Booking deletion
- Positive and negative API scenarios

## Tools Used
- Postman
- TestRail
- GitHub
- Jira

- 05-bug-reports

## Test Artifacts
